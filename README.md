# Cafe Appointment Booking UI
Second task for my UI/UX Internship.

## 🎯 Objective
Design a mobile-first booking experience for a local café to reduce scheduling confusion and improve customer experience.

## 🛠️ Tech Stack
- HTML5 / CSS3
- Tailwind CSS (CDN)
- JavaScript (State management for screen transitions)

## 📱 Features
- High-conversion Landing Screen
- Service/Zone Selection (Window Booth, Quiet Zone, etc.)
- Real-time Slot Selection
- Booking Confirmation
